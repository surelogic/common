Stop worrying about Java concurrency, start exploiting it

SureLogic's concurrency-focused analysis tools for Java and Android help you to create and maintain code that is safe and reliable. The tools help you understand shared state, diagnose and fix race conditions, and diagnose potential deadlock in your Java programs and Android apps. The tools also help you to avoid (or diagnose and fix) many other code hazards. Three tools: JSure (verification), Flashlight (dynamic), and Sierra (heuristic); provide broad analysis coverage during development, through testing, and into production.
